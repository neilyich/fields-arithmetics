# Core Fields Arithmetics Project
## Import (maven)
```xml
<dependency>
    <groupId>neilyich.fields</groupId>
    <artifactId>field-arithmetics</artifactId>
    <version>1.0</version>
    <scope>system</scope>
    <systemPath>${project.basedir}/lib/field-arithmetics-1.0.jar</systemPath>
</dependency>
```
